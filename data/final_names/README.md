# Final Names Lists

This folder contains the curated production name lists used by the renderer.

## Files
- `unified_popular_names_3000.txt`: final 3000-name list (alphabetical, one name per line).
- `unified_popular_names_3000_scored.csv`: same list with source/score metadata.
- `names_data_report.json`: source schema and merge/report snapshot used to build the list.

## Note
Generated render outputs stay in `output/` and should not be committed.

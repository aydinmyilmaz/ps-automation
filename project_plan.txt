# Photoshop Local Batch Name Renderer

## Product & Technical Specification (Phase 1)

---

# 1. Problem Statement

Photoshop uzmanı tarafından hazırlanmış, yüksek boyutlu (~500MB) bir PSD template dosyası üzerinden, farklı isimler kullanılarak otomatik PNG görseller üretmek istiyoruz.

Amaç:

* Manuel olarak her isim için PSD açıp text değiştirmek yerine
* Tamamen lokal çalışan
* Photoshop Desktop kullanan
* Otomatik, tekrarlanabilir
* Non-technical kullanıcı tarafından tek tıkla çalıştırılabilir
  bir batch üretim aracı geliştirmek.

---

# 2. Primary Goal (Phase 1)

Aşağıdaki çıktıyı elde etmek:

* Photoshop PSD template **tek sefer açılacak**
* İçindeki belirli bir text layer (`NAME_TEXT`) her iterasyonda güncellenecek
* Her isim için ayrı PNG export edilecek
* PNG’ler `output/` klasörüne yazılacak
* İşlem sonunda PSD kaydedilmeden kapanacak

Phase 1 kapsamı:

* 10 adet hardcoded test ismi
* Excel entegrasyonu yok
* Tek text layer
* Fit-to-box yok
* Tek PSD template

---

# 3. System Context

### Ortam

* OS: Windows
* Photoshop: Desktop sürüm (2023–2026 arası olabilir)
* Çalışma şekli: Lokal, offline

### Roller

#### Designer (Photoshop Uzmanı)

* `template.psd` oluşturur
* İçinde en az 1 text layer vardır
* Layer adı birebir: `NAME_TEXT`

#### Automation Developer

* Python runner geliştirir
* JSX script üretir
* `.exe` paketler
* Designer’ın tek tıkla çalıştırmasını sağlar

---

# 4. Core Architectural Decision

## CRITICAL DECISION: PSD tek sefer açılacak

500MB PSD her isim için tekrar açılırsa:

* Performans düşer
* Disk I/O artar
* Photoshop unstable olabilir

Doğru yaklaşım:

```
Photoshop Open
    └── Open template.psd (1 kez)
            └── Loop:
                  ├── Reset history state
                  ├── Set NAME_TEXT
                  ├── Export PNG
            └── Close without saving
```

Bu mimari:

* En stabil
* En performanslı
* En az memory fragmentasyonu yapan yöntem

---

# 5. Technical Architecture

## 5.1 High-Level Flow

```
run_batch.exe
    ↓
Python runner
    ↓
Generate JSX dynamically (inject names + paths)
    ↓
Call Photoshop
    ↓
Photoshop executes JSX
    ↓
PSD opened once
    ↓
Loop over names
    ↓
Export PNG files
```

---

# 6. Detailed Component Design

---

## 6.1 Python Runner

### Responsibilities

* Template var mı kontrol et
* Output klasörü oluştur
* Photoshop.exe path’ini bul
* JSX script oluştur (inject names)
* Photoshop’a script’i çalıştır
* Hata logla

### Why Python?

* Packaging kolay
* `.exe` üretilebilir
* Gelecekte Excel desteği kolay
* Photoshop path discovery yapılabilir

---

## 6.2 JSX Script

### Responsibilities

* Dialogları kapat
* template.psd aç
* NAME_TEXT layer’ını bul
* Base history state al
* names array üzerinde dön
* Her iterasyonda:

  * history reset
  * text set
  * safe filename oluştur
  * PNG export
* Save etmeden kapat

---

## 6.3 Layer Targeting Strategy

Phase 1:

```
doc.artLayers.getByName("NAME_TEXT")
```

Known Limitation:

* Eğer layer group içindeyse recursive search gerekir.
* Phase 2’de eklenebilir.

---

## 6.4 Export Strategy

Export method:

```
ExportOptionsSaveForWeb
PNG-24
Transparency true
Quality 100
```

Why not SaveAs?

* SaveAs bazı PSD konfigürasyonlarında overwrite uyarısı çıkarabilir.
* SaveForWeb genelde batch’te daha stabil.

---

# 7. Data Model (Phase 1)

Hardcoded:

```python
NAMES = [
    "Charlotte",
    "Olivia",
    ...
]
```

Output filename sanitization:

* Only `[a-zA-Z0-9_-]`
* Others → `_`

---

# 8. Error Handling Strategy

## Photoshop Not Found

* Net error mesajı
* Kullanıcıya candidates path listesi göster

## template.psd Not Found

* Exit with explicit message

## Layer Not Found

* JSX içinde try/catch
* Alert suppressed, log yazılmalı

## Popup Risk

Mitigation:

```
app.displayDialogs = DialogModes.NO;
```

Designer checklist:

* Font missing olmamalı
* Color profile ask kapalı

---

# 9. Performance Considerations

### 500MB PSD

Cost breakdown:

* Open: pahalı (bir kez)
* Export: nispeten ucuz
* Text change: çok ucuz

Expected Behavior:

* 10 isim: çok hızlı
* 1000 isim: lineer artış

Parallelization?

* Önerilmez.
* Photoshop tek instance stabil.
